部署步骤总览：
========================================

第一步：部署前端到 Netlify
- 文件夹：netlify-deploy/
- 时间：5-10分钟
- 说明：见 docs/超简化部署步骤.md

第二步：准备后端代码到 GitHub
- 文件夹：backend/
- 时间：5-10分钟
- 说明：见 docs/Netlify部署指南.md

第三步：部署后端到 Render.com
- 代码：从 GitHub 读取
- 时间：5-10分钟
- 说明：见 docs/Netlify部署指南.md

第四步：连接前后端
- 操作：修改 profile.html 中的 API 地址
- 时间：2分钟
- 说明：见 docs/Netlify部署指南.md

========================================
现在开始第一步：
========================================

1. 打开 netlify-deploy/ 文件夹
2. 访问 https://app.netlify.com/signup
3. 注册/登录账号
4. 点击 "Add new site" → "Deploy manually"
5. 将 netlify-deploy/ 文件夹拖拽到浏览器窗口
6. 等待部署完成

========================================
完成第一步后，继续：
========================================

阅读 docs/Netlify部署指南.md 文件
按照文档中的"后端服务部署"部分操作
